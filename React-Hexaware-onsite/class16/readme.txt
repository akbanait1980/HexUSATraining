# Context API
- context proide a way to pass data through the component tree without having to pass props manually at
every level

- in react we passed data in top-down approach vie props, it is difficult sometime for central type of
props that may required in multiple component

- context provides a simple way to pass data between components without passing explicitly through
every level of the component tree

# how to use context
there are two main steps to use react context into the application

1. setup a context provider and define the data which you want to store
2. use a context consume wheneven you need the data from the store
